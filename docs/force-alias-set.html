force-alias-set Task

Description

Sets an alias that the Salesforce CLI can use for various commands and tasks.

It executes the force:alias:set command.

Parameters

This task has no specific attribute apart from the attributes used by every ant-sfdx task.

Parameters specified as nested elements

alias

Specifies aliases to be set, sent as key=value pairs to the Salesforce DX CLI command. See property for usage guidelines.
